**局限性**

- 当前版本是通过坐标编码，固定了x轴间隔，变化y轴来做变异，只能说很不智能，并且现在都只是二维的，三维的情况变异更加复杂
- 时间复杂度高，规划慢，类似的启发式算法还是适用于优化的场景
- 对于GA, PSO之类大差不差的启发式算法，对它们的态度是把它们看成一种工具，数学解析解永远是最优的。但是当解析解没法算或者你不会算的时候，可以利用这些工具去解决一些问题，给你提供一些解决的依据。
